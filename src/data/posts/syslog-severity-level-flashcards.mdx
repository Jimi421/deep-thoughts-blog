# 📝 Syslog Severity Level Flashcards

# 

**Q:** What is Syslog severity level **0**?

**A:** **Emergency** – System unstable.

**Q:** What is Syslog severity level **1**?

**A:** **Alert** – Immediate action required.

**Q:** What is Syslog severity level **2**?

**A:** **Critical** – Failure in the system’s primary application.

**Q:** What is Syslog severity level **3**?

**A:** **Error** – Preventing proper system function.

**Q:** What is Syslog severity level **4**?

**A:** **Warning** – Potential issue, action needed to prevent error.

**Q:** What is Syslog severity level **5**?

**A:** **Notice** – Unusual event, not an error.

**Q:** What is Syslog severity level **6**?

**A:** **Informational** – Normal operation message, no action required.

**Q:** What is Syslog severity level **7**?

**A:** **Debug** – Developer-level detail for troubleshooting.
